While Loops

While loops in JavaScript are the same as python. They will repeat the execution of code provided the set condition is met. The syntax for While loops in JavaScript is shown below.

while (condition) {
code to execute;
code to execute;
code to execute;
}



function While1Function() {
var i=0;

while (i<=10) {
document.getElementById("WhileLoop1").innerHTML+="The i is "+i+"<br>";
i++; //increases i by 1, equivalent to i=i+1
}
}

For Loops

For loops in JavaScript are different than python. Instead of reapeating the execution of code a set number of times (example: for x in range (6)), they will repeat the execution of code based on a condition. The syntax for For loops in JavaScript is shown below.

for (intial value; condition; increase/decrease) {
code to execute;
code to execute;
code to execute;
}



function For1Function() {


for (var i=0; i<=10; i+=2) {
document.getElementById("ForLoop1").innerHTML+="The i is "+i+"<br>";

}
}
