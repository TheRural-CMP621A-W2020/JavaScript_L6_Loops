Student Exercise(s)

  1. Create a loop that will print out the numbers from 30 to 90 with increments of 3; 30, 33, 36, .................,87, 90
    Do this with a while loop and then try it with a for loop.
  2. Create a loop that will print out the numbers for 100 down to 25 with decrements of 5; 100, 95, 90, ..........., 35, 30, 25
    Do this with a while loop and then try it with a for loop.
  3. Create a list/array with 8 names in it. Use a for loop to print out names each on a seperate line.
    For example:
    Henry
    Sally
    Mike
    Jenny
    etc.
